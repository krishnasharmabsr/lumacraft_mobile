# LumaCraft V2 - Product Requirements Document

## Vision

A memory-efficient, lightning-fast video editor leveraging on-device ML and 100% free processing stack (FFmpeg).

## Target Audience

Mobile content creators needing quick trimming, AI-based cutouts, and reliable exporting without subscription walls (supported via AdMob).

## Core Features

1. **Import & Preview:** Seamless video loading with high-performance playback.
2. **Editing Engine (V2):** robust trimming and effect application pipeline.
3. **AI Processing:** On-device cutout/blur with memory-safe ML models.
4. **Export:** Reliable FFmpeg-based rendering with visual diagnostics.
5. **Monetization:** Integrated but unintrusive AdMob + Google Play Billing (Phase 3).
